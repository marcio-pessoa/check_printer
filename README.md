# check_printer
Nagios plugin

## Usage

### Sintax
``` bash
check_printer <IP_ADDR COMMUNITY RESOURCE WARNING CRITICAL>
```

### Example
``` bash
check_printer 192.168.0.2 public CYAN 30 15>
```
